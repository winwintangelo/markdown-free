# 免费！Markdown 一键转长图，公众号/小红书排版神器（无需注册·本地处理不上传）

## 你是不是也遇到过

用 ChatGPT、Obsidian 或 GitHub 写好的 Markdown 笔记，想发出来的时候——

- **公众号、小红书不支持 Markdown**，直接粘贴格式全乱，只能截图，还糊。
- **转 PDF 中文变成豆腐块（□□□）**，用 Pandoc 还要自己配字体，太折腾。
- **粘进 Word 表格和代码块全乱套。**
- 很多在线工具**要注册**，或者导出的图片**带水印/logo**。

而且把公司文档、客户资料传到不明来路的网站，心里也不踏实。

## 解决方案：Markdown Free（纯浏览器·免注册）

[**Markdown Free**](https://www.markdown.free/zh-Hans) 是一个在浏览器里就能把 Markdown 转成 **长图 / PDF / Word (DOCX) / EPUB** 的免费工具。

- ✅ **中文完美支持**，不会乱码、不会豆腐块，无需配字体。
- ✅ **无需注册、无需安装**，打开网页拖进去就行。
- ✅ **文件不上传** — 图片在你自己的浏览器里本地渲染，Markdown 内容不会发到服务器，公司文档也放心。
- ✅ **无水印、无广告、完全免费。**

## 重点：Markdown 转长图，专为公众号/小红书排版

1. 打开 [markdown.free/zh-Hans](https://www.markdown.free/zh-Hans)
2. 拖入 `.md` 文件，或直接粘贴 Markdown 文本
3. 预览确认后，点 **「转图片 (PNG)」**
4. 长文可选择导出 **一张长图**，或 **分段图片 ZIP**（超长内容更清晰）
5. 直接上传到公众号后台 / 小红书，**排版格式全部保留**
