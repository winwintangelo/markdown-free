# 【登録不要・無料】Markdown を PDF / Word / 画像に30秒で変換する（日本語 文字化けなし・保存なし）

## こんな経験 ありませんか？

ChatGPT や Obsidian、GitHub の README で書いた Markdown を、いざ人に渡そうとすると——

- **PDF にすると日本語が文字化け（□□□）する。** Pandoc でフォント設定（`--pdf-engine=xelatex -V mainfont="Noto Sans CJK JP"`）を毎回やるのが面倒。
- **Word に貼り付けると表やコードブロックの書式が崩れる。**
- **note や Zenn、X（旧Twitter）は Markdown をそのまま貼れない** ので、結局スクショを撮る羽目に。
- 変換サイトの多くは**登録が必要**だったり、**出力にロゴ（透かし）が入る**。

仕事のメモや社外に出す資料を、素性のわからないサイトにアップロードするのも気が引けます。

## 解決策：Markdown Free（ブラウザだけ・登録不要）

[**Markdown Free**](https://www.markdown.free/ja) は、ブラウザだけで Markdown を **PDF / Word (DOCX) / 画像 (PNG) / EPUB** に変換できる無料ツールです。

- ✅ **日本語 完全対応** — ひらがな・カタカナ・漢字が文字化けしません。フォント設定は不要。
- ✅ **登録・インストール不要** — ページを開いてドラッグするだけ。
- ✅ **ファイルは保存されない** — HTML/画像はブラウザ内（クライアントサイド）で生成。Markdown はサーバーに送信されません。機密資料でも安心。
- ✅ **透かしなし・広告なし・完全無料。**

## 使い方（約30秒）

1. [markdown.free/ja](https://www.markdown.free/ja) を開く
2. `.md` ファイルをドラッグ＆ドロップ（またはテキストを貼り付け）
3. プレビューを確認して、**「PDF に」「Word (DOCX) に」「画像 (PNG) に」** のいずれかをクリック
4. すぐにダウンロード完了

> （ここに実際のスクリーンショットを2〜3枚）

## 個人的におすすめ：Markdown →「画像」変換

意外と知られていませんが、**Markdown を1枚の画像（長い画像）に変換**できます。

- note・Zenn・X・Instagram など、**Markdown を貼れない場所に、書式を保ったまま画像として投稿**できる
- スクショと違って、**端末の解像度に合わせて文字がくっきり**
- 長い記事は「1枚の長画像」か「分割ZIP」を選べる

→ [Markdown を画像 (PNG) に変換する](https://www.markdown.free/ja/markdown-gazou-henkan)
