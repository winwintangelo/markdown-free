# 회원가입 없이 마크다운을 PDF·Word·이미지로 변환하는 무료 사이트 (한글 안 깨짐, 파일 저장 안 함)

## 이런 적 없으신가요?

ChatGPT나 Obsidian, GitHub README에 마크다운으로 정리해두고 남에게 전달하려는 순간—

- **PDF로 만들면 한글이 깨져서 □□□(두부 글자)로 나온다.** Pandoc으로 폰트(`Malgun Gothic`, `Noto Sans KR`)를 매번 설정하기 번거롭다.
- **Word에 붙여넣으면 표·코드블록 서식이 무너진다.**
- velog·브런치·인스타처럼 **마크다운을 그대로 못 붙이는 곳**엔 결국 캡처를 뜬다.
- 변환 사이트 대부분이 **회원가입을 요구**하거나 **결과물에 워터마크**가 박힌다.

회사 문서나 민감한 메모를 출처 모를 사이트에 업로드하는 것도 찜찜하고요.

## 해결책: Markdown Free (브라우저만, 회원가입 X)

[**Markdown Free**](https://www.markdown.free/ko) 는 브라우저에서 마크다운을 **PDF / Word (DOCX) / 이미지 (PNG) / EPUB** 로 변환하는 무료 도구입니다.

- ✅ **한글 완벽 지원** — 글자가 깨지지 않습니다. 폰트 설정 불필요.
- ✅ **회원가입·설치 불필요** — 페이지 열고 드래그하면 끝.
- ✅ **파일이 저장되지 않음** — 이미지/HTML은 브라우저 안(클라이언트)에서 생성되고, 마크다운 내용은 서버로 전송되지 않습니다. 사내 문서도 안심.
- ✅ **워터마크 없음 · 광고 없음 · 100% 무료.**

## 사용법 (약 30초)

1. [markdown.free/ko](https://www.markdown.free/ko) 접속
2. `.md` 파일을 드래그 앤 드롭 (또는 텍스트 붙여넣기)
3. 미리보기 확인 후 **「PDF로」「Word (DOCX)로」「이미지 (PNG)로」** 클릭
4. 즉시 다운로드

> (여기에 실제 스크린샷 2~3장)

## 추천 기능: 마크다운 → 이미지 변환

의외로 잘 안 알려졌는데, **마크다운을 한 장의 이미지(긴 이미지)로 변환**할 수 있습니다.

- velog·브런치·인스타처럼 **마크다운을 못 붙이는 곳에 서식 그대로 이미지로 게시**
- 캡처와 달리 **기기 해상도에 맞춰 글자가 또렷**
- 긴 글은 「한 장의 긴 이미지」 또는 「분할 ZIP」 선택 가능

→ [마크다운을 이미지 (PNG)로 변환](https://www.markdown.free/ko/markdown-imiji-byeonhwan)
